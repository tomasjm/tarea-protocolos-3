EL PROGRAMA NO HA PODIDO SER TESTEADO EN LAS RASPBERRYS

```git clone https://github.com/tomasjm/tarea-protocolos-3```

```cd tarea-protocolos-3```

```make```

Luego, para ejecturar los programas, en una terminal:

```make run1```

En otra terminal

```make run2```

En otra terminal

```make run3```
